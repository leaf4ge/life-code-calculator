# 五行學説

※人在三界內，活在五行中
※人法天，天法地，地法道，道法自然——老子
※五 = 是金（往內凝聚）、水（往下流）、木（往外擴散）、火（往上竄）、土（居中）五種物質。

行 = 運動變化，運行不息的意思
※相生 = 互相滋生、促進、增長
相克 = 互相製約、管製、克製
※人要健康要有三通：腸道要通、經絡要通（肝膽）、血管要通
※觀念要通，才能做到未病先知
※五行分陰陽，陽為看得見的、表層的（如皮膚）；陰為隱藏的、體內的（如五臟六腑）

## 【金 (1, 6)】（主肺，肺與大腸相表裏）

*金過旺容易便秘，尤其火也多更容易（因為金火旺，水容易不足）
*金過旺容易多愁善感，悲傷
*金過旺容易有上呼吸道、扁桃體、肺部問題，如咳嗽、哮喘、鼻炎
*金過旺容易磕磕碰碰，受傷（因為金是刀）
*金為 0 容易得哮喘、肺癌

## 【水 (2, 7)】（主腎，腎藏經，泌尿系統，由內分泌）

*腎好什麼都好，腎不好沒那麼多福氣
*水過旺容易晚年骨頭壞死（尤其名字有綁腿磁場、13 組有桃花號）
*水過少容易腰疼，女性經血少、血色暗、經期短
*水過少女容易有婦科問題，男容易有男科問題
*水平衡情商高、記憶力好、皮膚水靈靈的
*水不平衡和另一半感情也容易不好，欲望低
*水為財，水過少財富少
*水為 0 容易得尿毒癥、婦科癌癥

## 【木 (4, 9)】（主肝膽，主疏泄）

*肝好皮膚就好，因為排毒好
*木過旺容易偏頭痛，尤其是熬夜過後
*木過旺人容易木訥（尤其水少水生不動，木容易枯死），經絡不通，心情不好（披彩衣可以讓心情好，打通經絡）
*木過少眼睛容易有毛病，如近視、散光、幹澀
*木過少指甲容易有豎痕
*木為 0 容易得脂肪肝、肝癌、膽結石

## 【火 (3, 8)】（主心，心主血脈，主小腸）

*火過旺血管容易不通，因為血黏度高（川字紋很深的代表心血管有堵塞）
*火過旺，內火旺，容易冒青春痘
*火過旺容易得口腔潰瘍
*火過旺睡眠不好，因為心律不齊，人容易著急，容易失眠，經常做奇怪的夢
*火過旺發火之後容易腰骨疼（尤其火多水少）
*火過旺愛爭強好勝、風風火火，要停下來、要平靜
*火過旺傷身邊的人，火氣容易上來
*火為 0 容易得心梗、心臟病、腦梗

## 【土 (5)】（主脾胃，主運化，免疫器官）

*土生金，土弱則財運不旺（土弱+不得食+財庫小+沒有 6 = 掙多少都守不住）
*土過少思慮重，睡眠差
*土過少消化不好，要麼容易胖，要麼容易瘦，如胃虛、胃潰瘍
*土過少容易口臭（尤其是土少火多）
*土水過少身體機能會很差，年紀大了身體會萎縮
*土為 0 容易有三高/三低，血脂、血壓、血糖
*土為 0 經常怨氣重，因為吃不好睡不好，生活不如意
*土為 0 容易得突發性糖尿病、嚴重的胃病、甲狀腺疾病
*土為 0 且金多，容易得強直性脊椎炎
*土為 0 最不能忽視

---
**情感**：水多土 0，水容易泛濫、到處亂竄，身體不知道哪裏會出問題，沒有方向。金多土 0，沒有土生不了金。火多土 0，火燒完沒有土落地，無處為家（尤其是 3, 8 號人）。

### 【自身】
*自身強+財庫大+五行有 0，有錢但享不了

### 【財富子女】
*財富子女為 0 = 要麼財富子女二選一，要麼兩個都沒有問題，但自己享不了錢福、享不了孩子福
*子女財富弱 = 財庫小，沒錢孩子很好很乖，有錢孩子就鬧心花錢

### 【事業伴侶】
*事業伴侶為 0 = 想要夫妻恩愛、感情諧和、白頭偕老很難，事業伴侶二選一，因為五行是 0 所以自己身體也會 有問題

### 【官鬼疾病】
*官鬼疾病為 0 =
1. 在家庭當中地位低
2. 沒有官運，說話沒有分量，小人也多
3. 自身也弱一輩子病歪歪，小病不斷，藥罐子；自身強的平常不生病，一病要人命
4. 一輩子犯口舌是非各種官司/一輩子不打官司，一打就傾家蕩產
*克我的位置，不打任何招呼，隨時就來，毫無抵抗之力
*流年官鬼疾病為 0 同樣會出事，尤其流年走 7，不知有序或無常

### 【父母貴人】
*父母貴人為 0 = 父母幫不上，也沒有貴人幫，難依靠家裏，晚年窮困（因為未來財富為 0），百分六七十父親 先離開，父母身體容易出問題，靠近父母父母容易生病
*父母貴人為 0 = 一生的努力最後一場空
